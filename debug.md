2026-04-16 16:08 UTC
- 问题: 目录选择仍然基于 GPU 服务器路径，和浏览器标注工具的使用预期不符，用户无法直接操作自己电脑上的图片目录与标签目录。
- 原因: 之前的实现沿用了服务端工作区模型，把浏览器端目录选择误当成服务器目录浏览，而不是本地文件系统访问。
- 解决: 前端改为基于浏览器 File System Access API 的本地目录模式，支持用户直接选择本地图片目录和本地标签目录；标签读取和保存改为浏览器端本地文件读写，SAM 改成临时上传当前本地图片到后端推理。
